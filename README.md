# Homebridge RTL 433 

This is an work in progress!

## Development

You can run Rollup in watch mode to automatically transpile code as you write it:

```sh
  yarn run dev
```

Running:

```sh
DEBUG=* homebridge/bin/homebridge -D -U config -P rtl
```
